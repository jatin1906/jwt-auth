# Laravel JWT Authentication & Books CRUD API

A mini backend system developed using Laravel, MySQL, and JWT Authentication.

This project includes:
- JWT Authentication
- Books CRUD APIs
- Pagination & Search
- Soft Delete
- Validation
- Secure REST APIs

---

# 🚀 Features

## Authentication Module
- User Registration
- User Login
- JWT Token Authentication
- Protected Profile API

## Books CRUD Module
- Add Book
- Fetch All Books
- Fetch Single Book
- Update Book
- Delete Book (Soft Delete)

## Additional Features
- Pagination
- Search Functionality
- Validation
- JSON Responses
- Reusable Middleware
- Secure Authentication

# 🛠 Tech Stack

- PHP
- Laravel 12
- MySQL
- JWT Auth
- REST API
